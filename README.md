# Makeup Hobby Landing Page 💄

A simple interactive landing page built using HTML, CSS, and JavaScript.

This project showcases my front-end development skills through a beauty-themed webpage that highlights makeup as a creative hobby and form of self-expression.

>Features
-Clean and aesthetic design
-Image gallery layout using CSS Grid
-Hover animations on images
-Dark mode theme toggle
-Responsive centered layout
-External information link for additional reading

>Technologies Used
- HTML5
- CSS3
- JavaScript

>What I Learned

Through this project I practiced:

- Structuring webpages with HTML
- Styling layouts with CSS
- Using CSS Grid
- Creating hover effects and transitions
- Manipulating the DOM with JavaScript
- Implementing a dark mode feature

>Future Improvements:

- Add mobile responsiveness
- Include makeup category filters
- Add more interactive JavaScript features
- Create a makeup look recommendation quiz
